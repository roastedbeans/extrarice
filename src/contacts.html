About the Project


ExtraRise was developed by the team for the purpose of monitoring and forecasting the user’s income and expenses with the required data for the calculations. This project covers the topic for Interpolation, Extrapolation and Polynomial Approximation (Lecture 6) of CPE 3108 and it focuses on the use the lagrange method of interpolation. The project is still on its first phase of the development, hence, the functionality will be limited and more features are yet to be included on the next update.

Please see user guide attached with the folder.
